Books API Documentation

Method URL Body Description
GET /books Get all books
GET /books?perPage=3&page=2 Get books with pagination
POST /books { "name": "Book Name", "category": "Category", "author": "Author", "info": "Book Info", "price": 29.99 } Insert new book
POST /users { "name": "User Name", "email": "user@example.com", "password": "password" } Insert new user
POST /users/login { "email": "user@example.com", "password": "password" } Login
GET /users/usersList Get all users list (admin only)
DELETE /books/6569121039b9b242ebddae57 Delete book (admin or creator only)
PUT /books/6569109e39b9b242ebddae51 { "name": "Updated Book Name", "category": "Updated Category", "author": "Updated Author", "info": "Updated Book Info", "price": 39.99 } Edit book (user creator or admin only)
