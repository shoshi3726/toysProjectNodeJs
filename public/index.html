BOOKS API

Books Routes

Method URL Body Description
GET /books none Get a list of all books.
Optional queries:
  • perPage : you can select the amount of obj per one page.
  • page : you can select the num of pages you want.
    		•
    GET /books?perPage=3&page=2 none Get books with pagination. Specify perPage and page numbers as query parameters.
    GET /books/single/{book_id} none Get single book by his ID.
    GET /books/search/?{search word} none Get searched book, by its name or info, optional to add number of books perpage and page number
    GET /books/category/{catName} none Get books by category, optional to add number of books perpage and page number
    GET /books/prices/?{min=min_limit}&?{max=max_limit} none Get books by category, optional to add number of books perpage and page number
